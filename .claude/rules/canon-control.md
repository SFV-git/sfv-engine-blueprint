# CANON CONTROL RULE

Before touching ANY file, read the STATUS tag.

- STATUS: CANON → you may build from this file
- STATUS: UNCONFIRMED → do not touch, do not build
- STATUS: FUTURE → do not touch, not current phase
- STATUS: RESEARCH → reference only, never build from
- STATUS: REJECTED → never revisit without Will's approval
- STATUS: FOR HUMAN REVIEW → wait for Will's decision

If a file has no STATUS tag, stop and report it.
Never create files without a status tag.
