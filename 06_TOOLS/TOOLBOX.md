# TOOLBOX

Complete reference of every tool considered for SFV Engine.
See TOOL_STATUS.md for current status of each.

---

## AI INFRASTRUCTURE
- Ollama: local model runner, free, self-hosted
- Open WebUI: browser interface for Ollama
- OpenClaw: open-source autonomous AI agent, 100+ skills, local
- Claude API: Anthropic, high-value tasks only
- Whisper: OpenAI transcription, run locally

## WORKFLOW ORCHESTRATION
- n8n: primary workflow tool, self-hostable, visual
- Docker: container environment for stable deployments

## DATABASE / BACKEND
- Supabase: Postgres + auth + storage, open source Firebase alternative
- Postgres: underlying database

## FRONTEND / DASHBOARD
- Vercel: deployment platform for dashboard
- 21st.dev: UI component library
- Obsidian: blueprint brain and documentation

## REMOTE ACCESS
- Tailscale: secure private network between nodes

## MEDIA PROCESSING
- FFmpeg: video/audio processing
- Lightroom Classic: photo batch editing
- Premiere Pro: video editing
- Captions app: AI captions for reels

## DELIVERY
- Pixieset: client photo galleries
- WeTransfer: current delivery (temporary)

## SOCIAL / SCHEDULING
- Later: IG scheduling, handles multiple accounts [UNCONFIRMED — not chosen yet]
- Buffer: alternative to Later [UNCONFIRMED]

## GRAPHICS
- Canva API: template-based graphics for Level 6+ accounts [UNCONFIRMED]

## RESEARCH / SCRAPING
- Apify: web scraping, trend research, competitor analysis
