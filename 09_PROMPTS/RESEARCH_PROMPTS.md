# RESEARCH PROMPTS

Prompts for R&D Terminal research workflows.

---

## TOOL EVALUATION PROMPT
```
Evaluate [TOOL NAME] for use in a photography production system.
Return:
- What it does
- Cost (free/paid/per-use)
- Does it run locally or cloud-only?
- Relevant use cases for: ingest, culling, export, delivery, scheduling
- Verdict: YES / NO / MAYBE with reason
```

## TREND RESEARCH PROMPT
```
Research current Instagram content trends for [NICHE: fitness trainer / athlete / etc.]
Return:
- Top 5 content formats performing well right now
- Hook styles working in this niche
- Posting frequency norms
- What's oversaturated
Keep it practical. No generic advice.
```
