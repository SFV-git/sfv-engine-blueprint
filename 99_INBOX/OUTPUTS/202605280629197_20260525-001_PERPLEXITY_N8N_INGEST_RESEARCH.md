Your system's hardware (Ryzen 9 9900X, RTX 5080, 32GB RAM) and storage layout (C: vault, D: branch media, E: active ingest) are well-suited for a hybrid pipeline that combines **local automation** with **lightweight AI** for non-critical tasks. Here's how to break down the pipeline into components and assign them to tools:

---

### **1. Ingestion from SD/USB Drives**
**Tools**: **File watchers** (e.g., PowerShell, Python `watchdog`, or n8n)  
- **Why**: Monitor SD/USB drives for new files using file system events.  
- **Implementation**:  
  - Use **PowerShell** or **Python** scripts with `watchdog` to detect new files.  
  - **n8n** can trigger workflows when files are added to a directory (via the "File" node).  
- **Example**:  
  ```powershell
  # PowerShell script to monitor a directory
  $watcher = New-Object System.IO.FileSystemWatcher
  $watcher.Path = "D:\Ingest"
  $watcher.Filter = "*.*"
  $watcher.NotifyFilter = [System.IO.NotifyFilters]::LastWrite
  $watcher.Changed += { Write-Host "File added: $($EventArgs.Name)" }
  $watcher.EnableRaisingEvents = $true
  ```

---

### **2. Naming Conventions**  
**Tools**: **Local scripts** (Python, PowerShell) or **n8n**  
- **Why**: Apply consistent naming rules (e.g., `YYYYMMDD_HHMMSS_<source>_<filename>`).  
- **Implementation**:  
  - Use a **Python script** with `os.rename()` or **PowerShell** with `Rename-Item`.  
  - **n8n** can use the "Function" node to generate filenames based on metadata (e.g., timestamp, source device).  
- **Example**:  
  ```python
  import os
  for filename in os.listdir("D:\Ingest"):
      new_name = f"20240405_120000_SD_{filename}"
      os.rename(os.path.join("D:\Ingest", filename), os.path.join("D:\Ingest", new_name))
  ```

---

### **3. Deduplication via MD5 Checksum**  
**Tools**: **Checksum tools** (e.g., `md5sum`, `hashdeep`, or custom scripts)  
- **Why**: Prevent duplicate files from being processed.  
- **Implementation**:  
  - Use **`hashdeep`** (command-line tool) to generate MD5 hashes for all files in the ingest directory.  
  - Compare hashes against a database (e.g., SQLite or JSON file) to skip duplicates.  
  - **n8n** can trigger a checksum node to verify files before routing.  
- **Example**:  
  ```bash
  # hashdeep command to generate MD5 hashes
  hashdeep -r D:\Ingest > D:\Ingest\checksums.txt
  ```

---

### **4. Routing to Branch Folders**  
**Tools**: **n8n** or **local scripts**  
- **Why**: Move/copy files to branch folders based on metadata (e.g., file type, source).  
- **Implementation**:  
  - **n8n** can use the "File" node to move files to specific directories (e.g., `D:\Branch\Videos`, `D:\Branch\Photos`).  
  - **PowerShell** can use `Move-Item` with conditional logic.  
- **Example**:  
  ```powershell
  # PowerShell script to route files to branch folders
  $source = "D:\Ingest"
  $destination = "D:\Branch\Videos"
  Get-ChildItem $source | Where-Object { $_.Extension -eq ".mp4" } | Move-Item -Destination $destination
  ```

---

### **5. AI Review for Tagging/Captioning**  
**Tools**: **Lightweight local AI models** (e.g., Hugging Face Transformers, OpenCV, or ONNX models)  
- **Why**: Avoid high-cost AI services like Claude by using pre-trained local models.  
- **Implementation**:  
  - Use **Hugging Face's `blip`** or **`clip`** models for captioning and tagging.  
  - Run models on the **RTX 5080** using **PyTorch** or **ONNX Runtime**.  
  - **n8n** can trigger a Python script to process files with AI.  
- **Example**:  
  ```python
  # Using Hugging Face's BLIP model for captioning
  from transformers import BlipProcessor, BlipForConditionalGeneration
  import torch

  processor = BlipProcessor.from_pretrained("Salesforce/blip-image-captioning-base")
  model = BlipForConditionalGeneration.from_pretrained("Salesforce/blip-image-captioning-base").to("cuda")

  def generate_caption(image_path):
      image = Image.open(image_path).convert("RGB")
      inputs = processor(images=image, return_tensors="pt").to("cuda")
      outputs = model.generate(**inputs)
      return processor.decode(outputs[0], skip_special_tokens=True)
  ```
